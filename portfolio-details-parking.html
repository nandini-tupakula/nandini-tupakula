Parking Management System

The Parking Management System is a desktop-based Python application designed to automate the daily operations of vehicle parking in an organized and efficient manner. It facilitates secure admin login, real-time vehicle entry and exit tracking, intelligent slot management, and comprehensive report generation in both on-screen and downloadable PDF formats.

Built using Python and Tkinter for the GUI, MySQL for database storage, and FPDF & ReportLab for PDF generation, this project simulates a real-world parking lot environment suitable for apartments, offices, malls, or small commercial complexes.
