Bank Management System

The Bank Management System is a desktop-based banking application that emulates the key operations of a real-world bank. It is developed using JavaFX for the front-end user interface, MySQL for the backend relational database, and JDBC for seamless connectivity between the two. The system offers an interactive GUI and is designed to ensure secure, efficient, and user-friendly banking experiences.

    User Management:
The system includes a multi-step customer onboarding process. New users can register through a structured three-phase signup flow implemented using Signup.java, Signup2.java, and Signup3.java. Once registered, users can securely log in through the Login.java interface. Proper validation and user feedback mechanisms are included to enhance user trust and ease of use.

    Account Operations:
After logging in, users gain access to various banking operations. They can deposit funds (Deposit.java), withdraw money (Withdrawl.java), and perform quick transactions through the FastCash feature (FastCash.java). Each function is designed with input validation, error handling, and database updates for real-time execution.

    Balance & Statement Services:
Customers can check their current account balance using BalanceEnquiry.java. The system also provides a mini statement feature (mini.java) to display recent transaction summaries, helping users track their financial activity.

    PIN & Security Management:
Security is a core feature of this application. Users can change their transaction PIN using the Pin.java module. This functionality ensures that account holders can manage and secure their credentials effectively.

    Database Handling:
All backend database operations are managed through a dedicated database connector class Connn.java. It encapsulates all JDBC logic, ensuring efficient data transactions and reducing redundancy across the application.

    Application Control Flow:
The main user interface and overall navigation between modules are controlled via main_Class.java, which serves as the central hub of the application. It ensures smooth transitions between operations and provides a cohesive user experience.
